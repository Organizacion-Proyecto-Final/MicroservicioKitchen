<div align="center">
  <h1>🍳 Microservicio de Cocina</h1>
  <h3> Orquesta el flujo de trabajo dentro de la cocina y el estado de preparación de las órdenes. </h3>
</div>

* **Gestión de Órdenes de Cocina:** Recepción de nuevos pedidos y desglose en ítems (platos) a preparar.
* **Control de Estados:** Seguimiento del progreso de cada ítem y cambios de estados.
* **Límites de Concurrencia:** Configuración del número máximo de platos que la cocina puede preparar simultáneamente, optimizando el tiempo y evitando la sobrecarga.
* **Notificaciones en Tiempo Real:** Integración con **SignalR** para notificar inmediatamente a las pantallas de la cocina cuando entra un nuevo pedido o cambia el estado de una orden
